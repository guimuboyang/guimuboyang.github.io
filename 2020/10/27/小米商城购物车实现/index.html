小米商城购物车实现

小米商城购物车

由于在练习购物车案例时,实现复选框处于选中状态才进行计算功能时纠结了一段时间,所以写下来巩固一下

首先贴代码

function getsum() {
    let sum = 0;
    let count = 0;
    let allcount = 0;
    //从checkbox出发来找到需要的内容
    $(".select:checked").parent().siblings(".sumprice").each(function (i, ele) {
        sum += parseFloat($(ele).text());
    });
    //赋值给要显示的元素
    $(".allsum").text(sum);
    $(".select:checked").parent().siblings(".counter").find(".input").each(function (i, ele) {
        count += parseInt($(ele).val());
    });
    $(".count").text(count);
    $(".input").each(function (i, ele) {
        allcount += parseInt($(ele).val());
    })
    $(".allcount").text(allcount);
}

通过以上代码就可以实现通过复选框的勾选状态来改变需要计算的商品件数
