清除浮动的四种方法

清除浮动的四种方法

1.额外标签法(隔墙法)

在最后一个浮动的子元素后面添加一个空的块级元素并且给这个块级元素设置
clear:both

2.给浮动的父元素添加 overflow:hidden

3.after 伪元素方法

例如:

.clear:after{
    content:"";
    display:block;
    height:0;
    clear:both;
    visibility:hidden;
}
.clear{/*兼容IE6,7*/
    *zoom:1;
    }

4.双伪元素法

例如:

.clear:before,.clear:after{
    content:"";
    display:table;
}
.clear:after{
    clear:both;
}
.clear{/*兼容IE6,7*/
    *zoom:1
}

主要掌握第三种和第四种方法
